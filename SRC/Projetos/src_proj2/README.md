## Instalação em ambiente virtual

1. Crie o ambiente virtual:
```
python3 -m venv venv
```

2. Ative o ambiente virtual:
```
source venv/bin/activate
```

3. Instale as dependências:
```
pip install -r requirements.txt
```

## Execução

```
python3 script/sampleScript.py
```
